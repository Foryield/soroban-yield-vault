# @foryield/stellar-onboarding

D2 DFNS onboarding bricks for the Soroban yield vault: **provision** a DFNS Stellar wallet, build a transaction **envelope**, **submit** it, and run the full **onboard** flow end to end. Testnet only.

## Setup

```bash
cp .env.example .env
# fill in the DFNS values (provided out of band), then:
npm install
```
